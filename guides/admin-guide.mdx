This guide covers all administrative functions in App Courier, from managing daily operations to monitoring performance metrics.

## Accessing the Admin Panel

After logging in as an administrator, you'll see the main admin panel with four key modules:

- **Dashboard** - View analytics and performance metrics
- **Clientes** (Customers) - Manage customer accounts
- **Encomiendas** (Packages) - Track and manage all shipments
- **Motorizados** (Drivers) - Manage delivery drivers

You can access these modules by tapping the corresponding icons in the bottom navigation bar or by selecting them from the main panel cards.

## Managing Packages (Encomiendas)

### Viewing All Packages

<Steps>
  <Step title="Navigate to Packages">
    Tap the **Encomiendas** icon in the bottom navigation or select the Encomiendas card from the main panel.
  </Step>
  
  <Step title="Filter by Date and Branch">
    Use the date range filter to select a specific time period. Select a branch (sucursal) from the dropdown to filter packages by location.
  </Step>
  
  <Step title="Search Packages">
    Use the search bar to find packages by remito number, sender name, recipient name, or status.
  </Step>
  
  <Step title="View Package Details">
    Tap any package in the list to view complete details including tracking history, images, and delivery status.
  </Step>
</Steps>

<Note>
The package list updates in real-time and shows the most recent status for each shipment.
</Note>

### Creating a New Package

<Steps>
  <Step title="Start New Package">
    From the Encomiendas screen, tap the **+** (add) floating action button in the bottom-right corner.
  </Step>
  
  <Step title="Enter Package Details">
    Fill in the package information:
    - Package type (Sobre, Pequeño Paquete Documento, Pequeño Paquete Mercancía, or Encomienda Postal)
    - Weight and dimensions
    - Origin and destination locations
  </Step>
  
  <Step title="Add Sender Information">
    Enter sender (remitente) details:
    - Document type (DNI or RUC)
    - Document number
    - Full name
    - Phone number
  </Step>
  
  <Step title="Add Recipient Information">
    Enter recipient (destinatario) details with the same information fields as the sender.
  </Step>
  
  <Step title="Select Delivery Type">
    Choose the delivery type and confirm pickup/delivery addresses.
  </Step>
  
  <Step title="Review and Create">
    Review all information and tap the create button to register the package in the system.
  </Step>
</Steps>

<Tip>
The system automatically calculates shipping costs based on weight, dimensions, and distance between origin and destination.
</Tip>

### Assigning Drivers to Packages

<Steps>
  <Step title="Select Package">
    From the package list, tap on a package that needs driver assignment.
  </Step>
  
  <Step title="Access Assignment">
    Tap the **Asignar Motorizado** (Assign Driver) option from the package details screen.
  </Step>
  
  <Step title="Choose Driver">
    Select an available driver from the list of drivers in your branch.
  </Step>
  
  <Step title="Confirm Assignment">
    Confirm the assignment. The driver will immediately see this package in their assigned deliveries.
  </Step>
</Steps>

### Viewing Package History

To view the complete status history of a package:

1. Open the package details
2. Tap **Historial Estados** (Status History)
3. Review all status changes with timestamps

### Viewing Package Images

To view uploaded package images:

1. Open the package details
2. Tap **Imágenes Encomienda** (Package Images)
3. View all photos associated with the package

## Managing Customers (Clientes)

### Viewing Customer List

<Steps>
  <Step title="Navigate to Customers">
    Tap the **Clientes** icon in the bottom navigation or select the Clientes card showing the total number of registered customers.
  </Step>
  
  <Step title="Search Customers">
    Use the search bar to find customers by name or document number.
  </Step>
  
  <Step title="View Customer Details">
    Tap any customer to view their complete profile and shipping history.
  </Step>
</Steps>

### Adding a New Customer

<Steps>
  <Step title="Start New Customer">
    From the Customers screen, tap the **+** floating action button.
  </Step>
  
  <Step title="Enter Customer Information">
    Fill in the customer details:
    - Document type (DNI or RUC)
    - Document number
    - Full name
    - Email address
    - Phone number
    - Address
  </Step>
  
  <Step title="Save Customer">
    Review the information and tap save to create the customer account.
  </Step>
</Steps>

### Editing Customer Information

<Steps>
  <Step title="Select Customer">
    Find and tap the customer you want to edit.
  </Step>
  
  <Step title="Access Edit Mode">
    Tap the edit icon or **Editar Cliente** option.
  </Step>
  
  <Step title="Update Information">
    Modify any customer details as needed.
  </Step>
  
  <Step title="Save Changes">
    Tap save to update the customer information.
  </Step>
</Steps>

### Viewing Customer Shipping History

To view all packages sent or received by a customer:

1. Open the customer profile
2. Scroll to the shipping history section
3. View all associated packages with their current status

## Managing Drivers (Motorizados)

### Viewing Driver List

<Steps>
  <Step title="Navigate to Drivers">
    Tap the **Motorizados** icon in the bottom navigation or select the Motorizados card.
  </Step>
  
  <Step title="View Active Drivers">
    See all drivers assigned to your branch with their current status.
  </Step>
  
  <Step title="View Driver Details">
    Tap any driver to view their profile and delivery statistics.
  </Step>
</Steps>

### Viewing Driver Delivery History

<Steps>
  <Step title="Select Driver">
    Tap on a driver from the list.
  </Step>
  
  <Step title="Access History">
    Tap **Historial Encomiendas** (Delivery History).
  </Step>
  
  <Step title="Review Performance">
    View all packages delivered by this driver, including completed and pending deliveries.
  </Step>
</Steps>

<Note>
Driver performance metrics help you identify top performers and optimize delivery assignments.
</Note>

## Dashboard and Analytics

The Dashboard provides real-time insights into your branch's performance.

### Viewing Dashboard Metrics

<Steps>
  <Step title="Access Dashboard">
    Tap the **Dashboard** icon in the bottom navigation.
  </Step>
  
  <Step title="Select Time Period">
    Choose from three filter options:
    - **Diario** (Daily) - Today's metrics
    - **Semanal** (Weekly) - Current week's metrics
    - **Mensual** (Monthly) - Current month's metrics
  </Step>
  
  <Step title="Review Key Metrics">
    The dashboard displays:
    - Total packages for the selected period
    - Pending packages (not yet delivered)
    - Delivered packages
    - Package type distribution (Sobres, Documentos, Mercancías, Postales)
    - Total revenue (Ingreso Total)
  </Step>
</Steps>

### Understanding Dashboard Charts

The dashboard includes:

- **Status Cards**: Quick view of pending vs. delivered packages
- **Shipping Type Chart**: Bar chart showing distribution of package types
- **Revenue Card**: Total income generated from shipments

<Tip>
Use the weekly and monthly views to identify trends and plan resource allocation accordingly.
</Tip>

## Branch Management

Your assigned branch (sucursal) is displayed on the main admin panel. All operations, including package management, customer management, and driver assignments, are scoped to your branch.

## Account Settings

Access your account settings from the menu drawer:

### Opening the Menu

1. Tap the **menu icon** (hamburger menu) in the top-left corner
2. View your profile information at the top of the drawer

### Available Options

- **Mis datos** (My Profile) - View and edit your administrator profile
- **Manual de ayuda** (Help Manual) - Access the administrator help manual PDF
- **Soporte** (Support) - Contact support for assistance
- **Cerrar Sesión** (Logout) - Sign out of your account

<Note>
For security, always logout when using a shared device.
</Note>

## Troubleshooting

### Cannot Find a Package

- Check if the correct branch is selected in the filter
- Verify the date range includes when the package was created
- Try searching with different terms (remito, sender, or recipient)

### Driver Not Appearing in Assignment List

- Verify the driver is assigned to your branch
- Check if the driver account is active
- Refresh the driver list by returning to the main panel

### Dashboard Showing No Data

- Ensure the selected date range is correct
- Verify your branch has packages registered in that period
- Try switching between Diario, Semanal, and Mensual filters
