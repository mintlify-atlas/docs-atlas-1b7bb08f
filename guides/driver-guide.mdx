This guide covers all driver functions in App Courier, from viewing assigned deliveries to updating package status and tracking your performance.

## Accessing the Driver Panel

After logging in as a driver (motorizado), you'll see the main driver panel with two key modules:

- **Dashboard** - View your delivery statistics and performance metrics
- **Encomiendas** - View and manage your assigned deliveries

You can switch between these modules using the bottom navigation bar.

## Viewing Your Assigned Deliveries

The Encomiendas module shows all packages assigned to you for delivery.

### Accessing Your Delivery List

<Steps>
  <Step title="Navigate to Deliveries">
    Tap the **Encomiendas** (Packages) icon in the bottom navigation bar.
  </Step>
  
  <Step title="Select Date Range">
    Use the date range filter to view deliveries for specific days:
    - The filter defaults to today's date
    - Tap the date fields to select start and end dates
  </Step>
  
  <Step title="Select Branch">
    If you work in multiple branches, select the branch from the dropdown filter.
  </Step>
  
  <Step title="Search Deliveries">
    Use the search bar to find specific packages by:
    - Remito number
    - Sender name
    - Recipient name
    - Current status
  </Step>
  
  <Step title="View Query Results">
    Tap the **Consultar** (Query) button to load the filtered delivery list.
  </Step>
</Steps>

<Note>
The system only shows packages that have been assigned to you by an administrator. If you don't see a package, check with your branch administrator.
</Note>

### Understanding the Delivery List

Each package in your assigned deliveries shows:

- **Remito Number** - Unique package identifier
- **Sender (Remitente)** - Name and contact of the person sending the package
- **Recipient (Destinatario)** - Name and contact of the person receiving the package
- **Current Status** - The package's current delivery status
- **Delivery Address** - Where the package needs to be delivered

## Managing Package Deliveries

### Viewing Package Details

<Steps>
  <Step title="Select Package">
    Tap any package from your assigned deliveries list.
  </Step>
  
  <Step title="Review Information">
    View complete package details:
    - Package type and weight
    - Pickup and delivery addresses
    - Sender and recipient contact information
    - Special delivery instructions
    - Package status history
  </Step>
  
  <Step title="View Package Images">
    If available, tap to view photos of the package taken during registration.
  </Step>
</Steps>

### Updating Package Status

As you progress through the delivery process, update the package status:

<Steps>
  <Step title="Open Package Details">
    Select the package you're updating from your delivery list.
  </Step>
  
  <Step title="Access Status Update">
    Tap the status update or edit option (available based on your current stage).
  </Step>
  
  <Step title="Select New Status">
    Choose the appropriate status:
    - **En tránsito** - Package picked up and in transit
    - **En sucursal destino** - Package arrived at destination branch
    - **En reparto** - Out for delivery to recipient
    - **ENTREGADO** - Successfully delivered
  </Step>
  
  <Step title="Add Details (if required)">
    Some status updates may require additional information:
    - Delivery confirmation (signature or photo)
    - Recipient name who received the package
    - Delivery notes
  </Step>
  
  <Step title="Save Update">
    Confirm the status update. The system will timestamp the change and notify relevant parties.
  </Step>
</Steps>

<Tip>
Update package status in real-time as you complete each delivery step. This keeps customers informed and helps administrators track progress.
</Tip>

### Contacting Sender or Recipient

If you need to contact the sender or recipient:

1. Open the package details
2. Note the phone number displayed for sender (remitente) or recipient (destinatario)
3. Use your phone to call the contact
4. If there are delivery issues, update the package notes with relevant information

<Note>
Always maintain professional communication when contacting customers. Clearly identify yourself as the courier driver and reference the remito number.
</Note>

## Delivery Best Practices

### Before Starting Deliveries

<Steps>
  <Step title="Check Assigned Packages">
    Review all packages assigned to you in the Encomiendas screen.
  </Step>
  
  <Step title="Verify Package Condition">
    Ensure all packages are in good condition before leaving the branch.
  </Step>
  
  <Step title="Confirm Contact Information">
    Check that recipient contact information is available for all deliveries.
  </Step>
  
  <Step title="Update Status to In Transit">
    Mark packages as "En tránsito" when you begin your delivery route.
  </Step>
</Steps>

### During Delivery

<Card title="Verify Recipient Identity">
  Always verify the recipient's identity matches the package information before delivery.
</Card>

<Card title="Handle with Care">
  Treat all packages carefully to prevent damage during transport and delivery.
</Card>

<Card title="Update Status Immediately">
  Update package status as soon as each delivery is completed.
</Card>

<Card title="Document Issues">
  If a delivery cannot be completed, note the reason in the package details and contact your administrator.
</Card>

### Completing a Delivery

<Steps>
  <Step title="Arrive at Delivery Location">
    Navigate to the delivery address shown in the package details.
  </Step>
  
  <Step title="Contact Recipient">
    Call the recipient if needed to confirm they're available to receive the package.
  </Step>
  
  <Step title="Verify Identity">
    Confirm the recipient's identity matches the package information.
  </Step>
  
  <Step title="Hand Over Package">
    Deliver the package to the recipient.
  </Step>
  
  <Step title="Get Confirmation">
    Obtain delivery confirmation (signature or photo as required).
  </Step>
  
  <Step title="Update Status to Delivered">
    Mark the package as "ENTREGADO" in the system with any required delivery notes.
  </Step>
</Steps>

## Handling Common Situations

### Recipient Not Available

<Steps>
  <Step title="Attempt Contact">
    Call the recipient using the phone number in the package details.
  </Step>
  
  <Step title="Wait Reasonable Time">
    Wait a reasonable amount of time for the recipient to respond.
  </Step>
  
  <Step title="Document Attempt">
    Add notes to the package about the failed delivery attempt.
  </Step>
  
  <Step title="Contact Administrator">
    Contact your branch administrator for instructions on next steps.
  </Step>
  
  <Step title="Update Package Status">
    Update status appropriately (may remain "En reparto" pending re-delivery).
  </Step>
</Steps>

### Address Issues

If you cannot find the delivery address:

1. Double-check the address in the package details
2. Call the recipient for clarification
3. Use your phone's map application for navigation assistance
4. Contact your administrator if the address is incorrect or incomplete
5. Document the issue in the package notes

### Package Damage

If you discover a package is damaged:

1. Do not deliver the damaged package
2. Take photos of the damage
3. Contact your administrator immediately
4. Document the damage in the package notes
5. Return the package to the branch

### Wrong Package Information

If package details don't match the physical package:

1. Do not complete the delivery
2. Contact your administrator for verification
3. Document the discrepancy
4. Return to branch if instructed

## Troubleshooting

### Cannot See Assigned Packages

- Verify you're logged in with the correct driver account
- Check that the date range filter includes today
- Ensure the correct branch is selected
- Tap **Consultar** to refresh the package list
- Contact your administrator if packages are missing

### Cannot Update Package Status

- Check your internet connection
- Verify you have permission to update the package
- Ensure you're selecting a valid status for the current stage
- Try logging out and back in
- Contact support if the issue persists

### App Running Slowly

- Close other apps on your device
- Check your internet connection speed
- Clear app cache if available
- Restart the app
- Ensure your device has sufficient battery
