This guide helps you use App Courier to request shipments, track your packages, and manage your account.

## Getting Started

After logging in to your customer account, you'll see the main customer dashboard with a package tracking search feature and access to key functions.

### Main Dashboard Features

The customer dashboard provides:

- **Quick Package Lookup** - Search for packages by remito number
- **Menu Access** - Tap the menu icon to access all customer features
- **Welcome Message** - Personalized greeting showing your name

## Tracking Your Packages

### Quick Package Search

<Steps>
  <Step title="Enter Remito Number">
    On the main dashboard, enter your package's remito number in the search field.
  </Step>
  
  <Step title="Tap Search">
    Tap the **Buscar** (Search) button to look up the package.
  </Step>
  
  <Step title="View Package Details">
    A popup will display the package information:
    - Current status (Estado)
    - Origin address (Origen)
    - Destination address (Destino)
    - Estimated delivery date (Fecha entrega)
    - Recipient name (Receptor)
  </Step>
  
  <Step title="Close Details">
    Tap **Cerrar** (Close) to return to the dashboard.
  </Step>
</Steps>

<Note>
You can only track packages where you are listed as the sender. The remito number is provided when you create a shipment request.
</Note>

## Requesting a New Shipment

Create shipment requests directly from the app, which will be processed by the courier service.

### Creating a Shipment Request

<Steps>
  <Step title="Open Menu">
    Tap the **menu icon** in the top-left corner of the dashboard.
  </Step>
  
  <Step title="Select Request Shipment">
    Tap **HAZ TU ENVÍO** (Request Shipment) from the menu.
  </Step>
  
  <Step title="Enter Package Information">
    Fill in the package details:
    - **Package Type**: Select from:
      - Sobre (Envelope)
      - Pequeño Paquete Documento (Small Document Package)
      - Pequeño Paquete Mercancía (Small Merchandise Package)
      - Encomienda Postal (Postal Package)
    - **Weight**: Enter package weight
    - **Dimensions**: Enter dimensions if calculating by volume
  </Step>
  
  <Step title="Select Origin and Destination">
    Choose pickup and delivery locations:
    - Select origin branch or district
    - Enter pickup address if using home pickup
    - Select destination branch or district
    - Enter delivery address
  </Step>
  
  <Step title="Enter Sender Information">
    Fill in your details as the sender (remitente):
    - Document type (DNI or RUC)
    - Document number
    - Full name
    - Phone number
  </Step>
  
  <Step title="Enter Recipient Information">
    Fill in the recipient's (destinatario) details:
    - Document type (DNI or RUC)
    - Document number
    - Full name
    - Phone number
  </Step>
  
  <Step title="Review Pricing">
    The system automatically calculates the shipping cost based on:
    - Package type and weight
    - Distance between origin and destination
    - Delivery type (branch pickup vs. home delivery)
  </Step>
  
  <Step title="Submit Request">
    Review all information and submit your shipment request.
  </Step>
</Steps>

<Tip>
Keep your remito number safe! You'll need it to track your package and for pickup/delivery verification.
</Tip>

### Understanding Package Types and Limits

Each package type has specific weight restrictions:

- **Sobre** (Envelope): Up to 0.5 kg
- **Pequeño Paquete Documento**: Up to 2 kg
- **Pequeño Paquete Mercancía**: Up to 10 kg
- **Encomienda Postal**: Up to 30 kg

<Note>
The system will validate your package weight against the selected type. Choose the appropriate type to avoid request errors.
</Note>

## Managing Your Shipment Requests

### Viewing Pending Requests

View shipment requests that haven't been processed yet:

<Steps>
  <Step title="Open Menu">
    Tap the menu icon and select **ENVÍOS PENDIENTES** (Pending Shipments).
  </Step>
  
  <Step title="Filter by Date">
    Use the date range filter to narrow down your search to specific periods.
  </Step>
  
  <Step title="Search Requests">
    Use the search bar to find requests by sender name, recipient name, or delivery type.
  </Step>
  
  <Step title="View Request Details">
    Tap any request to view complete details including pricing and status.
  </Step>
</Steps>

<Note>
Pending requests are shipments you've created but that haven't been assigned a remito number yet. Once processed by the courier service, they move to "Attended Requests."
</Note>

### Viewing Attended Requests

Check the status of processed shipment requests:

<Steps>
  <Step title="Open Menu">
    Tap the menu icon and select **ENVÍOS ATENDIDOS** (Attended Shipments).
  </Step>
  
  <Step title="Filter Results">
    Use date filters to find specific shipments.
  </Step>
  
  <Step title="View Shipment Status">
    Each attended shipment shows:
    - Remito number
    - Current status
    - Sender and recipient information
    - Tracking history
  </Step>
</Steps>

<Tip>
Attended requests have been processed and assigned remito numbers. You can now track them using the quick search on the main dashboard.
</Tip>

## Understanding Shipment Statuses

Your packages may have the following statuses:

- **Pendiente** - Package registered, awaiting pickup
- **En tránsito** - Package is being transported
- **En sucursal destino** - Package arrived at destination branch
- **En reparto** - Out for delivery
- **ENTREGADO** - Successfully delivered

    - Registered address
  </Step>

  </Step>
  
  <Step title="Save Changes">
    Tap save to update your profile.
  </Step>
</Steps>

<Note>
Keeping your contact information up-to-date ensures you receive important notifications about your shipments.
</Note>

## Account Security

### Logging Out

<Steps>
  <Step title="Open Menu">
    Tap the menu icon to open the drawer.
  </Step>
  
  <Step title="Select Logout">
    Tap **Cerrar Sesión** (Logout) at the bottom of the menu.
  </Step>
  
  <Step title="Confirm Logout">
    Confirm you want to sign out of your account.
  </Step>
</Steps>

<Tip>
Always logout when using the app on a shared or public device to protect your account information.
</Tip>

### Forgot Password

If you need to reset your password:

1. On the login screen, tap **Forgot Password**
2. Enter your registered email address
3. Check your email for the password reset code
4. Enter the code in the verification screen
5. Set your new password

## Troubleshooting

### Cannot Find My Package

- Verify you're entering the correct remito number
- Ensure you're logged in with the account used to create the shipment
- Check that the package has been processed (moved from Pending to Attended requests)
- Contact support if the remito number doesn't return results

### Shipment Request Failed

- Check that package weight matches the selected package type
- Verify all required fields are filled in
- Ensure origin and destination addresses are complete
- Verify internet connection is stable

### Cannot Submit New Request

- Make sure all required information is provided
- Check that phone numbers are in valid format
- Verify email address is correct
- Try refreshing the form and entering information again
